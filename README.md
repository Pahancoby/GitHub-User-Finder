# GitHub User Finder

## Автор
Разницын Илья

## Описание
GUI-приложение для поиска пользователей GitHub с использованием API.

## Как работает API
Используется GitHub API:
https://api.github.com/search/users?q=USERNAME

## Возможности
- Поиск пользователей
- Добавление в избранное
- Сохранение в JSON

## Запуск
python main.py

## Пример использования
1. Ввести "octocat"
2. Нажать "Поиск"
3. Добавить пользователя в избранное
